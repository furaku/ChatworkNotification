ChatworkNotification
======

## Overview
The application waits in the taskbar and monitors Chatwork messages, notifying you of unread.

## Requirement
* .Net Framework 4.8 ~

## Install
Sorry, no installer and no pakage. Please build from source files yourself.

Chatwork通知
